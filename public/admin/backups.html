How backups work
  • Daily snapshot taken automatically, kept for 14 days on the Railway volume.
  • Download button takes a fresh snapshot and downloads it to your computer — safe to do anytime, no lock on the live DB.
  • Cloud upload (Cloudflare R2 / AWS S3) runs automatically if S3_BUCKET is configured in Railway env vars.
  • See DEPLOY-BACKUPS.md in the repo for the full setup walkthrough.
